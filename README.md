# How to run the application

## Development server

1. Install the dependecies:
   ```bash
   $ yarn
   ```
2. Start the development server:
   ```bash
   $ yarn dev
   ```

## Build

1. Install the dependecies:
   ```bash
   $ yarn
   ```
2. Start the development server:
   ```bash
   $ yarn build
   ```

## Access the website

1. Access the website deployed on heroku: https://calckcal.herokuapp.com/.
